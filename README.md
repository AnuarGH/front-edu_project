## hq-front

### Technologies
React :+1: / Redux

### Usage:

Clone repository:
```
git clone https://github.com/alisheriskakov/hq-front
```

Go to repository folder:
```
cd learning-app-back
```

Install the dependencies
```
npm i
```

Run the server (default port: 3000)
```
npm start
```
